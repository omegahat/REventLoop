To install this package, you will need to have access to some of the header files in R that are not installed when R is installed on the system. So it is easiest to install the package from a version of R that is not installed but run directly from the directory in which it was built. However, all that is necessary is to specify the location of the directory containing the file IOStuff.h. That is found in the src/include/ directory of the R source tree. You can tell the package where this directory is in either of two ways.
  • You can set the environment variable R_INCLUDE_DIR to the name of the fully qualified src/include directory
    This is done as 
     setenv R_INCLUDE_DIR /wherever/R/source/is/src/include
    for the Csh/Tcsh shells or 
     export R_INCLUDE_DIR=/wherever/R/source/is/src/include
    for the Bourne shell (sh/bash).
  • You specify the directory via the --with-R-includes flag for the configuration file. 
     R CMD INSTALL --configure-args='--with-R-includes=/wherever/R/source/is/src/include' REventLoop
